# ToDo Let's

React, Vite ve Tailwind CSS kullanılarak geliştirilmiş, sürükle-bırak (drag & drop) özelliğine sahip Kanban tarzı bir görev yönetim uygulaması.

Proje; görevleri "Yapılacaklar", "Yapılıyor" ve "Tamamlandı" sütunları arasında taşıma, yeni görev ekleme, düzenleme ve silme özelliklerini içerir.

Projeyi bilgisayarınızda çalıştırmak için terminalde sırasıyla şu komutları yazın:

```bash
npm install
npm run dev
```
